
1. Для запуска программы установите JDK (Java Development Kit)
https://www.oracle.com/java/technologies/downloads/

2. Далее , с помощью консоли войдите в корнейвой каталог с програмой 
пример : 
cd E:\ArtecSystemsTasks\AristekSystemTask2\src

3. Для запуска программы введите следующий код 
java -jar AristekSystemTask2.jar -top-words 10 -name MoonStara

где
-top-words -- имя команды
-name -- имя команды
10 -- количество строк которые может вывести программа 
MoonStara -- имя исполнителя
